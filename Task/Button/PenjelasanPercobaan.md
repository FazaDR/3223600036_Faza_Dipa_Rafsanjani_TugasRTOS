Percobaan ini menunjukkan bagaimana dua tombol (button) dapat diproses secara independen menggunakan dua core prosesor pada ESP32. Masing-masing tombol dijalankan dalam task terpisah yang dipasangkan ke core berbeda (button 1 di core 0 dan button 2 di core 1). Setiap task membaca input tombol dengan mode INPUT_PULLUP dan mendeteksi perubahan dari tidak ditekan ke ditekan (falling edge) untuk melakukan toggle pada variabel status.
