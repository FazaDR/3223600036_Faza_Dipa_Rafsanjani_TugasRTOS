Percobaan ini mengontrol dua servo menggunakan dua core berbeda untuk menguji apakah keduanya dapat bergerak secara sinkron. 
Masing-masing servo dijalankan dalam task terpisah di Core 0 dan Core 1 dengan pola gerak, kecepatan, serta jeda waktu yang sama. Awalnya gerakan kedua servo tampak sinkron, namun seiring waktu terjadi pergeseran (drift) karena perbedaan kecil dalam waktu eksekusi antar-core, ketidaktepatan fungsi penundaan vTaskDelay(), serta variasi proses PWM pada tiap core. Hasilnya, percobaan ini membuktikan bahwa tanpa mekanisme penyamaan waktu, dua task di core berbeda tidak dapat mempertahankan sinkronisasi sempurna meskipun memiliki logika yang identik.
